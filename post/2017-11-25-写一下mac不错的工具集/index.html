写一下MAC不错的工具集

2017-11-25

今天写一下MAC开发常用的软件吧,如有雷同，请务必究

Go2Shell

这款软件可以很快的从当前文件夹打开控制台到当前的路径，可以放置小图标到finder的小图标，貌似现在不可以放置了

APN Tester

通知的测试工具，可以测试调试和发布的通知，好处是可以自动识别苹果返回原始字符串，自动带有通知字符串的格式，唯一缺点是容易崩溃，要手动选择通知证书

JSON Accelerator

生成JSON model类的工具，可以选取语言类型，可以设置是否支持ARC，貌似现在很多人都用JSON的第三方库，但是字段还要自己添加，很麻烦。这个只要自己复制JSON一键生成。如果以后字段修改，只要再次生成覆盖一下即可

Resizer

这个是由高清@2x生成一倍图，已经抛弃不用，改用下面的一款

PaintCode

这个是iOS开发的神器，可以做原型之后用代码生成，支持很多属性，可以自定义复杂的控件，可以自定义动画，还有更多的功能需要大家挖掘了

Cornerstore

这是一款SVN管理的工具，绝对是最强大的 比versions强大多了，不多解释

iFunBox

这个是安装ipa包用的，还有提起手机资源给手机输入资源，可以在发版本的时候自己先测试一下自己发的包是否正确

iTools

这个和ifunbox功能差不多 提供额外的资源下载

ColorSnapper

获取颜色 支持UIColor NSColor 十六进制网页格式等等

Charles

抓包获取请求接口的神器

Mark Man

标记UI 间距 大小 颜色 文本

SourceTree

Git管理工具神器

TextWrangler

这个是可以打开我们平时打不开的文件 神器

IconKit

制作图标的工具 只用1024x1024的 不过已经暂且不用了

FauxPas

这个是可以找到工程里面的错误信息 隐藏的错误 找到没用的文件 不支持的API等等 神器

Echo

调试接口 不如Paw

PhotoShop

自己可以制作图标等

JSONExport

JSON输出工具

Paw

网络调试工具 还支持输出AF 自带类的代码

Pngyu

png压缩图片大小

Brirfs

制作原型

Sketch

制作原图

Reveal

调试页面UI 可以使用FLEX代替

ImageOptim

压缩图片的大小 神器

xScope

UI 标记 苹果一直在用

iResize

生成一倍图 用prepo代理

Prepo

可以由@3x生成@2x @1x 由小图生成大图 可以生成图标 可以预览

AppCode2

可以和安装插件Xcode想提并论的开发工具 神器

EasyAPN

测试通知 缺点是不支持没格式化的token 不自带通知字段 好处不用选择通知证书

QuartzCode

做动画的神器 现在还没研究美白

Color Picker

取色

ImageAlpha

压缩透明度

ColorCube

颜色取色 能获取一张图所有的配色

PxCook

图标记

Sequel Pro

mysql查看工具

GIFBrewery

制作GIF动画提取GIF动画的图片

JSON Editor

用plist形式生成JSON

UIysses

markdwon编辑器

Mou

同上

dysm

这个是让符号化变成代码行的工具 可以用BugHD代替

SimPholders

快速打开模拟器的目录
