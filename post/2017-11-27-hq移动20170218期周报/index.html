6、注意 某些方法的 执行顺序

 if (self.testBlock) {
      self.testBlock();
 }
 [self dismissViewControllerAnimated:YES completion:^{}];

[self dismissViewControllerAnimated:YES completion:^{
      if (self.testBlock) {
          self.testBlock();
      }
}];

[self dismissViewControllerAnimated:YES completion:^{}];
if (self.testBlock) {
      self.testBlock();
}

上述三者在执行某个问题时，也许效果一样，第二种会有点卡顿，第一种和第三种基本一样，但是一定注意第一种的执行顺序，有时这样写会遇到某种意想不到的BUG，可能外部调用时也会写 [self dismissViewControllerAnimated:YES completion:^{}];, 但是实际上可能的是 那个 self 还没有被释放掉，而导致问题啦。

另外注意 第二种和第三种的方法不一样，至于使用哪一种看具体需求情况。

毕竟这个Block 里面的执行时间是在 viewDidDisappear 之后的。
