利用GitHub 的Plist连接生成的在线安装iPA包不弹出安装提示框

2017-11-27

问题描述

之前都是好好的，突然测试说我的下载页面原来安装不了。我尝试了一下，也是没有安装的提示。

在群里面问也没有得到有效的帮助，就把问题搁置在哪里了。

解决办法

今天让其他同事尝试竟然可以安装，我突然想起来了。我这个手机安装有 AppStore 的包，会不会是这个原因导致的。

删掉从 AppStore 的正式包之后，再次果然是可以弹出安装的对话框的。

起先我还有点怀疑是公司屏蔽了 GitHub 的网或者是 GitHub 屏蔽了我们的。
